
# Search Hints vs Prompt Phrasing

Separate two concepts:

Search hints
- abbreviations
- aliases
- spelling variants
- confusion queries

Prompt phrasing
- natural learner language
- instructional tone
- exam phrasing

Example:

Search hint:
"как найти СО"

Prompt phrasing:
"Как найти степень окисления в соединении K2Cr2O7?"
