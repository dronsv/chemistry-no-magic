
# Topic Layer Principles

## Definition

Topic = pedagogical unit of the course.

Examples:
- Ionic bond
- Covalent polar bond
- Oxidation state rules
- Molar mass

Topics represent what learners search for and what teachers explain.

## Architectural rule

Topic belongs to the pedagogical layer, not the ontology.

Ontology describes the chemical world.
Topics describe how we teach it.

## Responsibilities

Topics connect:

- theory modules
- competencies
- example entities
- task templates
- topic pages
- practice entrypoints
