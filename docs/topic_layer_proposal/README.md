
# Topic Layer – Unified Proposal

This package consolidates the architectural decisions discussed for the **Topic layer** in the chemistry platform.

Key idea:

Topic = pedagogical / search intent mapping layer.

It connects:

Ontology → Topics → Pages / Practice / SEO / AI

Topics are **not chemical entities** and **not SEO hacks**.  
They represent units of teaching and user intent.

Layers:

Core Ontology
- elements
- ions
- substances
- reactions
- processes

Pedagogical Layer
- competencies
- theory modules
- topics
- task templates

Delivery Layer
- topic pages
- practice profiles
- SEO shells
- RSS
- sitemap
- llms.txt
